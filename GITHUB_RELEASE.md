# Relax Time v0.4.0

## 🎉 首次發布

Relax Time 是一個專為 Windows 設計的時間管理工具，幫助您管理工作時間和休息時間，保護您的眼睛健康。

## ✨ 主要功能

### ⏱️ 智能計時器
- 可調整倒計時時間（以 5 分鐘為單位，預設 30 分鐘）
- Start / Pause / Stop 完整時間控制
- 大螢幕倒計時顯示，清晰易讀

### 👁️ 眼睛休息模式
- 工作時間結束後自動進入休息模式（5 分鐘）
- 自動最小化所有視窗，強制您休息
- 休息結束後自動恢復工作環境

### 🔄 循環模式
- 啟用循環模式後，休息結束自動重新開始工作計時
- 適合長時間工作場景，保持工作節奏

### 🔽 系統托盤整合
- 可隱藏到系統托盤（右下角）
- 點擊托盤圖標可在螢幕中央顯示視窗
- 托盤圖標顯示當前狀態和剩餘時間

### 🚀 開機自動啟動
- 支援開機自動啟動功能
- 可在主視窗中輕鬆設定
- 啟動時自動隱藏到托盤

### 🎨 用戶體驗
- 現代化、簡潔的界面設計
- 鬧鐘圖標，易於識別
- 支援繁體中文和英文介面

## 📦 下載

### 推薦：安裝程式
下載 `RelaxTime-Setup-0.4.0.exe` 並執行安裝

**優點：**
- 標準 Windows 安裝流程
- 可選擇安裝目錄
- 自動創建桌面和開始菜單快捷方式
- 支援開機自動啟動選項
- 完整的卸載功能

### 便攜版
下載 `RelaxTime.exe` 直接執行（無需安裝）

**優點：**
- 無需安裝，即下即用
- 適合臨時使用或試用

## 🚀 快速開始

1. **安裝或執行**
   - 使用安裝程式：下載 `RelaxTime-Setup-0.2.0.exe` 並安裝
   - 或直接執行：下載 `RelaxTime.exe` 雙擊運行

2. **開始使用**
   - 設定工作時間（預設 30 分鐘，可調整）
   - 點擊「開始」按鈕開始計時
   - 時間到後會自動進入休息模式（最小化所有視窗）

3. **進階功能**
   - 啟用循環模式：休息結束後自動重新開始
   - 托盤隱藏：工作時可隱藏視窗
   - 開機啟動：設定後每次開機自動運行

## 💡 使用技巧

- **循環模式**：適合需要長時間工作但想保持規律休息的場景
- **托盤隱藏**：工作時可以隱藏視窗，需要查看時點擊托盤圖標
- **開機啟動**：設定開機啟動後，每次開機都會自動運行（隱藏模式）

## 📋 系統要求

- Windows 7 或更高版本
- 不需要安裝 Python 或其他依賴

## 🛠️ 技術特點

- **MVC 架構**：清晰的代碼結構，易於維護
- **多線程設計**：計時器獨立運行，不阻塞界面
- **Windows 原生整合**：使用 Windows API 實現視窗管理
- **輕量級**：使用 Python + tkinter，資源占用低
