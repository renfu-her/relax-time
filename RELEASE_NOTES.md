# Relax Time v0.5.0 Release Notes

## 🎉 v0.5.0 更新

### 🆕 新增功能

#### 🎬 全螢幕倒數遮罩（多螢幕支援）
- 在倒數計時剩下 5 秒時顯示全螢幕透明黑色遮罩
- **多螢幕支援**：自動檢測所有顯示器，支援 Extend 模式（如 15.6" + 27" 外接螢幕）
- 為每個螢幕創建獨立的遮罩視窗，確保所有螢幕都被覆蓋
- 主螢幕顯示大號倒數數字：5, 4, 3, 2, 1
- 其他螢幕只顯示遮罩（不顯示數字）
- 倒數完成後自動關閉遮罩並進入休息模式
- 提供更好的視覺提示，讓用戶清楚知道即將進入休息

#### 🚫 功能調整
- **移除休息時間的遮罩**：從休息返回工作時不再顯示倒數計時和遮罩
  - 休息時間結束後直接恢復視窗，無需等待倒數
  - 提供更流暢的用戶體驗

## 🎉 首次發布（v0.3.0）

Relax Time 是一個專為 Windows 設計的時間管理工具，幫助您管理工作時間和休息時間，保護您的眼睛健康。

## ✨ 主要功能

### ⏱️ 智能計時器
- 可調整倒計時時間（以 5 分鐘為單位，預設 30 分鐘）
- Start / Pause / Stop 完整時間控制
- 大螢幕倒計時顯示，清晰易讀

### 👁️ 眼睛休息模式
- 工作時間結束後自動進入休息模式（5 分鐘）
- 自動最小化所有視窗，強制您休息
- 休息結束後自動恢復工作環境

### 🔄 循環模式
- 啟用循環模式後，休息結束自動重新開始工作計時
- 適合長時間工作場景，保持工作節奏

### 🔽 系統托盤整合
- 可隱藏到系統托盤（右下角）
- 點擊托盤圖標可在螢幕中央顯示視窗
- 托盤圖標顯示當前狀態和剩餘時間

### 🚀 開機自動啟動
- 支援開機自動啟動功能
- 可在主視窗中輕鬆設定
- 啟動時自動隱藏到托盤

### 🎨 用戶體驗
- 現代化、簡潔的界面設計
- 鬧鐘圖標，易於識別
- 支援繁體中文和英文介面

## 🛠️ 技術特點

- **MVC 架構**：清晰的代碼結構，易於維護
- **多線程設計**：計時器獨立運行，不阻塞界面
- **Windows 原生整合**：使用 Windows API 實現視窗管理
- **輕量級**：使用 Python + tkinter，資源占用低

## 📦 安裝說明

### 方式一：使用安裝程式（推薦）

1. 下載 `RelaxTime-Setup-0.5.0.exe`
2. 雙擊執行安裝程式
3. 按照安裝嚮導完成安裝
4. 安裝完成後可在開始菜單或桌面啟動程式

### 方式二：直接執行 exe

1. 下載 `RelaxTime.exe`
2. 雙擊執行即可使用（無需安裝）

## 🚀 快速開始

1. 設定工作時間（預設 30 分鐘）
2. 點擊「開始」按鈕開始計時
3. 時間到後會自動進入休息模式
4. 休息結束後可以繼續工作或啟用循環模式自動繼續

## 💡 使用技巧

- **循環模式**：適合需要長時間工作但想保持規律休息的場景
- **托盤隱藏**：工作時可以隱藏視窗，需要查看時點擊托盤圖標
- **開機啟動**：設定開機啟動後，每次開機都會自動運行（隱藏模式）

## 📋 系統要求

- Windows 7 或更高版本
- 不需要安裝 Python 或其他依賴
